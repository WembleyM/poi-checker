# POI Checker — статическая PWA для GitHub Pages

Файлы в этом архиве готовы к загрузке в корень репозитория GitHub. После включения GitHub Pages получите ссылку вида:
`https://<ваш-логин>.github.io/<название-репозитория>/`

PWA можно установить на Android («Установить приложение») и пользоваться ежедневно с иконки.
